DATA VISUALIZATION PROJECT USING PYTHON

Title: Data Visualization of Monthly Sales Using Python

Objective: To analyze and visualize monthly sales data using Python
libraries to understand trends and patterns in business performance.

Tools Used: - Python - Pandas - Matplotlib - Seaborn

Dataset (sales.csv):

Month, Sales Jan, 50000 Feb, 60000 Mar, 55000 Apr, 70000 May, 75000 Jun,
80000 Jul, 82000 Aug, 78000 Sep, 85000 Oct, 90000 Nov, 92000 Dec, 98000

Python Code Used:

1.  Load Data import pandas as pd df = pd.read_csv(“sales.csv”)
    print(df)

2.  Line Chart import matplotlib.pyplot as plt plt.figure(figsize=(8,5))
    plt.plot(df[“Month”], df[“Sales”], marker=“o”) plt.title(“Monthly
    Sales Trend”) plt.xlabel(“Month”) plt.ylabel(“Sales”) plt.grid(True)
    plt.show()

3.  Bar Chart plt.figure(figsize=(8,5)) plt.bar(df[“Month”],
    df[“Sales”]) plt.title(“Monthly Sales Comparison”)
    plt.xlabel(“Month”) plt.ylabel(“Sales”) plt.show()

4.  Heatmap import seaborn as sns pivot = df.set_index(“Month”)
    plt.figure(figsize=(6,5)) sns.heatmap(pivot, annot=True,
    cmap=“coolwarm”) plt.title(“Sales Heatmap”) plt.show()

Findings: - Sales generally increase throughout the year. - Highest
sales are in December. - Slight dip occurs in August. - Overall business
growth is positive.

Applications: - Helps companies analyze performance. - Supports better
business decisions. - Useful for forecasting future trends.
